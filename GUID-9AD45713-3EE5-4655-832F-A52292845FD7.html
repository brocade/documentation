Disabling MFNM mode

  1. Connect to the switch and log in using an account assigned to the admin role.
  2. Enter the ag --pgdelmodes command with the appropriate options to disable MFNM mode. In the following example, MFNM mode is disabled for port group 3. 
    switch:admin> ag --pgdelmodes 3 "mfnm"
Managed Fabric Name Monitoring mode has been disabled for Port Group 3
  3. Enter the ag --pgshow command to display the port group configuration. If disabled, "mfnm" should not display under PG_Mode for port 3.

    For more details on this command and its operands, refer to the Fabric OS Command Reference.

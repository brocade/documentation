Create a Read-Only UserÂ¶

This example shows how to create a Spectre user with read-only access privilege. First you create the role, and then you add the user.

  1. Click the Settings icon ( ), and then select General Settings > User and User Group Management.

  2. Create a read-only role.

    1. Select the Roles tab, and click the Add Role button ( ) in the top right corner of the page.

    2. Give the role a name.

    3. (Optional) Add a description and one or more tags.

    4. Select what privileges you want the user to have.

      By default, the Read access level is selected for all privileges.

      Scroll down to see additional privileges. In this example, all privileges are selected.

    5. Click Save.

  3. Add the user and assign the read-only role.

    1. Select the Users tab, and then click the Add User button ( ) in the top right corner of the page.

    2. Fill out the standard user form.

    3. In the Roles section, select the read-only role you created.

    4. In the AOR section, select the hosts and fabrics you want the user to see.

      In this example, All Fabrics is selected. Note that by default, the user account is activated.

    5. Click Save.

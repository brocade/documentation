Forward error correction support

Forward error correction (FEC) is a Fabric OS feature supported in the following configurations:

  • Between the AG device F_Port and a Brocade 16 Gbps Host Bus Adapter (HBA) port running version 3.2 or later firmware.
  • Between the AG device N_Port and F_Port on a Brocade 16 Gbps fabric switch or cascaded AG switch.

Consider the following imitations for FEC:

  • Supported on Brocade 16 Gbps platforms only.
  • Supported by Fabric OS 7.1.0 and later.
  • Enabled by default.
  • Disabled when Fabric OS is downgraded to an earlier release.
  • Supported on specific switch platforms in R_RDY mode or VC_RDY mode.

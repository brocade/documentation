Register the Server as Syslog and SNMP Trap Recipient
1. Click the Settings icon ( ), and then select Network Monitoring > Registration.
Figure 13. Syslog Registration page
2. Click the Syslog tab, and enter the syslog listening port number.
3. Select Auto register server as Syslog recipient to automatically register the server as a syslog recipient when new switches are discovered.
4. Click the SNMP Trap tab and similarly register the server as an SNMP trap recipient.

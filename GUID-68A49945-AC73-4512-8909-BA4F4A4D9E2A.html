Part Number: 53-1003937-02

FDMI support

Starting with Fabric OS 7.3.0, an AG device can register its N_Port with FDMI devices. Use the fdmishow command to display the device details in AG mode. The fdmishow command displays only the local devices. Remote device details are blocked.

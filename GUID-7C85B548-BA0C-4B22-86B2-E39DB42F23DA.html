AG mode without all PoD licenses

Prior to Fabric OS 7.3.0, the Brocade non-embedded switches (Brocade 300, 5100, 6505, and 6510) require all Ports on Demand (PoD) licenses to run in Access Gateway mode. However, starting with Fabric OS release 7.3.0, all PoD licenses are not required to run these switches in AG mode.

Consider the following points while running the switches in AG mode without PoD licenses:

  • By default, configured N_Ports will come up as disabled because a PoD license is not installed for those ports.
  • All F_Ports mapped to the default N-Port will come up as disabled with the "N-Port Offline for F-Port" message displayed.
  • You must manually configure N_Ports in the AG device using the portcfgnport command, and move the cable connections accordingly.
  • You can update the N_Port-to-F_Port mapping using the mapdel and mapadd commands.
  • You must install all PoD licenses to downgrade from Fabric OS 7.3.0 to any previous release.
